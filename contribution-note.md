# Contribution Note

This repository is based on an **individual course final project** for **ALY6030: Data Warehousing & SQL**.

My work in this repo includes:
- reviewing the raw sample structure
- reorganizing the raw file into dimension and fact tables
- creating the SQL database logic
- defining primary keys and foreign keys
- selecting referential actions
- creating the ERD submission
- writing the reporting queries
- preparing the report and portfolio-facing version

## Scope Note

This was a small academic project built from a course-provided sample dataset. The project is not meant to represent a production healthcare system. I am showing it here as a clean example of SQL, normalization, warehouse design, and reporting logic.
