# Contribution Note

This repository is based on an **individual course final project** for **ALY6030: Data Warehousing & SQL**.

My work in this repo includes:
- reviewing the raw course sample structure
- reorganizing the wide raw file into dimension and fact tables
- creating the MySQL schema logic
- defining primary keys and foreign keys
- selecting referential actions for the fact table relationships
- creating the ERD submission
- writing the SQL reporting queries
- preparing the report and portfolio-facing documentation

## Scope Note

This was a small academic project built from a course-provided sample dataset. It is not a production healthcare system, deployed PBM warehouse, automated ETL pipeline, or dashboard application.

The project is included as a clear example of SQL data modeling, fact/dimension design, and reporting logic on a small classroom sample.
